# 🎬 Simulación de Sala de Cine en C
Simulación de sala de cine en C que permite reservar y cancelar asientos, con validaciones robustas de entrada para prevenir errores del usuario y garantizar estabilidad del programa.

Este proyecto consiste en una simulación básica de gestión de una sala de cine desarrollada en lenguaje C. 

El sistema permite visualizar la sala, reservar asientos y cancelar reservas mediante una interfaz por consola. Aunque es un proyecto sencillo en funcionalidades, se enfoca especialmente en la validación robusta de datos para evitar errores comunes de entrada por parte del usuario.

## 🛠 Funcionalidades

- Visualización del estado actual de la sala
- Reserva de asientos disponibles
- Cancelación de reservas
- Salir

## 🔐 Validaciones implementadas

- Verificación del tipo de dato ingresado.
- Prevención de ingreso de letras en variables numéricas.
- Prevención de ingreso de números en variables tipo texto.
- Manejo de entradas inválidas sin que el programa se cierre
- Control de rangos válidos para filas y columnas
- Prevencion de desbordamiento de memoria.

## 🧠 Conceptos aplicados

- Arreglos bidimensionales (matrices)
- Manejo de memoria básica
- Validación de entrada y limpieza de buffer
- Programación estructurada
